# Binox Swap - BNB Smart Chain (BSC) DEX

Binox Swap is a decentralized exchange built on **BNB Smart Chain (BSC)** and part of the **BNB Chain ecosystem**.

This project uses **BNB**, **BNB Chain**, and **BSC infrastructure** to enable fast and low-cost token swaps.
